# 📊 راهنمای راه‌اندازی دیده‌بان بورس

این پروژه شامل ۲ بخش است:
- **api/** → سرور پروکسی (روی Vercel اجرا می‌شود، مشکل CORS را حل می‌کند)
- **public/index.html** → داشبورد دو-تب (دیده‌بان + گزارش مالی)

---

## مرحله ۱ — آپلود روی GitHub

1. وارد **github.com** شو
2. روی **New repository** بزن
3. اسم بگذار مثلاً: `bourse-dashboard`
4. **Public** انتخاب کن (برای Vercel رایگان لازم است)
5. روی **Create repository** بزن

### آپلود فایل‌ها از گوشی:
1. داخل ریپوی خالی، روی **uploading an existing file** بزن
2. تمام فایل‌های این پوشه (`api/`, `public/`, `package.json`, `vercel.json`) را drag & drop کن
   - **مهم:** ساختار پوشه‌ها باید حفظ شود (api/tsetmc.js نه فقط tsetmc.js)
3. پایین صفحه **Commit changes** بزن

---

## مرحله ۲ — اتصال به Vercel

1. برو به **vercel.com**
2. **Sign Up** → **Continue with GitHub** (همون حساب گیت‌هاب رو وصل کن)
3. بعد از ورود، روی **Add New → Project** بزن
4. ریپوی `bourse-dashboard` رو از لیست انتخاب کن → **Import**
5. در صفحه تنظیمات:
   - Framework Preset: **Other**
   - Root Directory: همون پیش‌فرض بمونه (`.`)
6. روی **Deploy** بزن

⏳ حدود ۳۰-۶۰ ثانیه طول می‌کشه.

---

## مرحله ۳ — اضافه کردن API Key (مهم!)

برای امنیت بیشتر، کلید API را به‌جای کد، در تنظیمات Vercel بگذار:

1. در پروژه Vercel → **Settings** → **Environment Variables**
2. اضافه کن:
   - **Name:** `BRSAPI_KEY`
   - **Value:** `BbIG8Hx9jkUhG4vzVcqAKSsXZZDSw8Wb`
3. **Save**
4. به تب **Deployments** برگرد → روی آخرین deployment کلیک کن → **Redeploy**

> اگه این مرحله رو رد کنی هم مشکلی نیست، چون کلید به‌صورت پیش‌فرض داخل کد هم هست — ولی بهتره از Environment Variable استفاده کنی.

---

## مرحله ۴ — باز کردن داشبورد

بعد از deploy، Vercel یک آدرس می‌ده شبیه:

```
https://bourse-dashboard-xxxx.vercel.app
```

این آدرس رو باز کن (از گوشی یا هر دستگاهی) — داشبورد باید مستقیم به API وصل بشه و داده واقعی نشون بده، چون این بار درخواست از **سرور Vercel** میره نه از مرورگر، و مشکل CORS از بین می‌ره.

## ⚠️ نکات مهم

### محدودیت نرخ (Rate Limit)
- BrsApi رایگان محدودیت تعداد درخواست داره
- داشبورد گزارش‌های کدال (ن‌۳۰/ن‌۱۰) رو فقط برای **۶۰ نماد برتر** (بر اساس ارزش بازار) می‌گیره تا به محدودیت نخوریم
- اگه خطای rate-limit دیدی، چند دقیقه صبر کن و دوباره امتحان کن

### آپدیت خودکار
- داشبورد هر **۲ دقیقه** خودش رو آپدیت می‌کنه
- می‌تونی با دکمه «↻ آپدیت» دستی هم بزنی

### اگه چیزی کار نکرد:
1. بررسی کن که ساختار فایل‌ها در گیت‌هاب دقیقاً همینه:
   ```
   bourse-dashboard/
   ├── api/
   │   ├── tsetmc.js
   │   └── codal.js
   ├── public/
   │   └── index.html
   ├── package.json
   └── vercel.json
   ```
2. در Vercel → **Deployments** → آخرین deploy رو باز کن → تب **Functions** → ببین `api/tsetmc` و `api/codal` ساخته شدن
3. در مرورگر، آدرس `https://your-app.vercel.app/api/tsetmc` رو مستقیم باز کن — باید JSON ببینی
